# 선택된 방향

## 이름
**Gajae Desk Ledger**

## 한 줄 정의
Blue Crab과 Red Claw를 한 쌍으로 섞지 않고, 각각 독립적인 작업대 인상으로 운영하는 가재 블로그 전용 운영 기록 시스템.

## 왜 이전 안이 아니라 이 방향인가
이전 안은 이름만 crab이었고 실제 색은 너무 중성적이거나, 두 테마를 한 화면 문법으로 섞어 읽히게 만들었다.
이번 방향은 gajae-code TUI의 blue-crab / red-claw 색을 그대로 끌어와서,
blue-crab은 blue 위주의 차가운 작업대,
red-claw는 red 위주의 뜨거운 작업대로 각각 따로 읽히게 만든다.

## 핵심 조합
- **Blue Crab**: abyss / deep navy / ocean / claw blue 중심
- **Red Claw**: ink / mantle / deep red / claw orange-red 중심
- **Proof layer**: evidence slab / receipt chips / sage verification
- **Signal layer**: theme accent / amber warning / clipped incident note

## 화면 인상
- 마케팅 랜딩보다 작업 기록
- 대시보드보다 현장 바인더
- 장식용 terminal보다 실제 evidence strip
- 캐릭터 브랜딩보다 작성자 서명

## 재질 언어
- 동일한 둥근 카드 복제 대신 서로 다른 문서 실루엣
- 큰 면은 종이, 작은 증거는 그라파이트
- 배지는 pill보다 stamp에 가깝게
- 중요한 경계는 색면보다 얇은 룰(line)과 rail로 해결

## 상호작용 원칙
- hover는 lift보다 정리감 우선
- glow, blur, neon 금지
- 색 대비보다 위계와 간격으로 우선 해결
