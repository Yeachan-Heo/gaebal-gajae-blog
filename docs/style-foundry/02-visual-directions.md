# 비주얼 방향 탐색

## Direction A — Warm Field Notebook
**비중: 60% / 기본 바탕**

### 시그널
- 크림색 종이, 연필/잉크, 얇은 구획선
- 공책·바인더·현장 메모 같은 정리감
- 과장된 브랜딩보다 문서성 우선

### 여기서 가져올 것
- 전체 캔버스 톤
- 카드 기본 재질
- 타이포 위계
- 섹션 간 여백 리듬

### 리스크
노트 감성이 지나치면 기술 운영 로그가 아니라 취향형 라이프스타일 에디토리얼처럼 보일 수 있다.

---

## Direction B — Terminal Proof Lab
**비중: 30% / 증거 레이어**

### 시그널
- 흑연색/그라파이트 계열 증거 블록
- 모노 타이포와 receipt 형식 메타
- verified / warning 신호를 장식이 아니라 상태로 사용

### 여기서 가져올 것
- proof block
- repo / metadata rail
- setup tip 재현 포인트
- 검증/번역 상태 레이블

### 리스크
터미널 감각을 장식처럼 남발하면 진짜 증거가 아니라 코스프레처럼 보인다.

---

## Direction C — Playful Incident Report
**비중: 10% / 제한된 강조**

### 시그널
- 스탬프, 사건 배지, clipped slip
- 약간의 랍스터 컬러 포인트
- 장난기가 아니라 작성자 서명 정도의 존재감

### 여기서 가져올 것
- incident 카드 변형
- 상태 스탬프
- reference slip 실루엣

### 리스크
이 레이어가 커지면 블로그의 신뢰도가 떨어지고 캐릭터 사이트처럼 보인다.

## 선택 규칙
새 화면은 반드시 **노트북이 먼저 읽히고**, 그 위에 **증거 블록이 보조**로 붙고, 마지막에 **가벼운 incident accents**가 찍혀야 한다.
어느 요소가 이 순서를 깨면 제거하거나 약화한다.
