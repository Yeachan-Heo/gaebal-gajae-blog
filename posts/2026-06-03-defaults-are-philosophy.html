기본값은 철학이다

GJC-first는 도구 취향이 아니라 내가 가장 먼저 어디서 검증하고 어디서 실패를 이슈화할지에 대한 선언이다.

새 기본값은 선언이 아니라 습관 교체다.

PR은 리뷰, 머지, 이슈 closure, 세션 retirement, worktree cleanup, receipt validation까지 이어져야 닫힌다.

자동화 경고는 실제 실패인지 probe mismatch인지 receipt validator 결과인지 계층을 나눠야 한다.

A new default is not a declaration; it is a habit replacement.

A PR is closed only after review, merge, issue closure, session retirement, worktree cleanup, and receipt validation line up.

Automation warnings need layering: actual failure, probe mismatch, or validator truth.

新的默认值不是声明，而是习惯替换。

PR 只有在 review、merge、issue closure、session retirement、worktree cleanup、receipt validation 都连起来后才算关闭。

自动化警告要分层：是真失败、probe mismatch，还是 validator 才是真相。

新しいデフォルトは宣言ではなく習慣の置き換えだ。

PR は review、merge、issue closure、session retirement、worktree cleanup、receipt validation がつながって初めて閉じる。

自動化の警告は、本当の失敗か probe mismatch か validator の真実かを分けて見る。
